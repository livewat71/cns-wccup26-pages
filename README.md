# cns-wccup26-pages

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅用于存储和展示一组独立的、静态的 HTML 页面。

## 项目简介

本仓库的主要目的是将多个独立的 HTML 页面集中存放，并提供简单的浏览和访问方式。这些页面可能包含信息展示、工具界面、演示页面等内容，每个页面均为独立的 HTML 文件，不依赖第三方服务或数据库。

## 目录说明

```
.
├── index.html          # 仓库主页（可选，用于导航）
├── page1.html          # 示例页面一
├── page2.html          # 示例页面二
├── assets/             # 公共资源文件夹（如 CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- 仓库根目录下的 `.html` 文件即为可独立访问的页面。
- `assets/` 文件夹用于存放页面共享的静态资源，保持文件整洁。
- 如有需要，可创建子目录组织页面，但建议保持路径简洁。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，可直接在浏览器中打开。
- 页面之间无强制依赖关系，可独立运行。
- 页面内容可能随时间更新或替换，历史版本可通过 Git 提交记录追溯。
- 本仓库不存储用户数据，不涉及动态内容生成。

## 维护说明

- 欢迎提交 Pull Request 添加新页面或改进现有页面，但请确保新页面不包含恶意代码或侵犯他人权益的内容。
- 提交前请检查 HTML 语法正确性，避免损坏页面结构。
- 保持 `assets/` 文件夹整洁，避免冗余资源。
- 如有重大变更，请在提交信息中简要说明。

## 许可

本仓库中所有页面内容（包括 HTML、CSS、JavaScript 等）默认采用 [MIT 许可证](LICENSE)，除非页面内另有明确声明。
